# Donnees electorales

Organisation par famille d'election.

## Dossiers

- `municipales/`
- `cantonales/`
- `departementales/`
- `regionales/`
- `legislatives/`
- `presidentielles/`

## Convention conseillee

Deposer les CSV directement dans le dossier du type d'election correspondant.

Exemple:

- `data/elections/municipales/2014-municipales-tour-1-ministere-interieur.csv`
- `data/elections/presidentielles/2022-presidentielles-tour-2-ministere-interieur.csv`

Si un meme scrutin a plusieurs fichiers source, garder une convention homogène sur `tour` et `source`.
